# 08 — 100% 整理頁裝飾

**用途**：`/shelf/[bookId]/compilation` 頁面頂部裝飾，及社交媒體分享預覽圖
**輸出路徑**：`public/illustrations/compilation-header.png`
**尺寸建議**：1200 × 630（OG / 社交分享比例）or 1080 × 1080（IG 方版）兩種建議都生

**這是分享行銷的核心 asset，視覺要打中**。

## 概念

一張像書本扉頁的水彩畫：中央是一本攤開的書，書頁上飛出許多便利貼，便利貼形成一個輕盈的螺旋或飄帶。底部有「— Folio —」浮水印位置。

## Prompt

```
A soft watercolor illustration designed for social media sharing: at the center,
a beautifully weathered open book lies face-up. Out of the open pages,

each note hand-drawn with rough rectangular edges, some tilted, some overlapping.
The notes form an organic upward spiral, suggesting "memories rising from a book."
Around the book, faint hand-drawn decorative elements: a tiny crescent moon, a delicate leaf, a small star,
a coffee cup steaming softly. Background is warm cream with a barely-visible paper grain.
Bottom of the frame has a thin horizontal line of pages, like the bottom of the Folio logo.

soft watercolor illustration, hand-painted texture, slightly grainy paper background,
warm parchment color palette (#F8F1E4 base, sage green #6B8064, lavender #9B8FB5,
dusty rose #E8C4BD, amber #D9B884 — all soft pastels, walnut #3D2F1F for thin outlines),
literary cozy mood, two crops to provide:
(A) horizontal 1200x630 composition for social Open Graph cards,
(B) square 1080x1080 composition for Instagram,
gentle bleeding edges, no harsh ink lines, no neon colors,
no figures, no text in image — text overlay added in code.
```

## Avoid

- 浮水印 / Folio 字（之後 code 加）
- 太亮的金色（會破壞水彩感）
- 滿版便利貼（要留呼吸空間）

## 變體想法

- 季節版：春（櫻花瓣）、夏（綠葉）、秋（落葉）、冬（小雪花）— 加進便利貼飛舞中
